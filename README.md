# GloscAI Images APP

使用原生安卓应用开发的一款生图AI；

## 功能
- 工程式的生图（设定好参数和提示词后，点击生成即可）
- 对话式的生图（可以和AI进行对话，AI会根据对话内容生成图片）
- 图片编辑（对图片实现拓展、修复、变换、局部修改等功能）
- 图片变换（对图片实现风格转换、图像增强、图像修复等功能）
- 接入自定义API（可以接入第三方的AI服务，或者自己搭建的AI服务）
- 图片管理（可以对生成的图片进行管理，支持分类、标签、搜索等功能）

## 技术栈
- Android原生开发（Java/Kotlin）
- OpenAI API（用于生成图片）
- Room数据库（用于图片管理）
- Retrofit（用于网络请求）
- Glide（用于图片加载）
